# style-dataset-lab

Comfy-backed visual dataset lab. Generate, curate, and export multimodal training data.

## What this is

A repo whose whole job is to become **trainable visual truth**. Every asset carries three things:

1. **Image pixels** — generated by ComfyUI with full provenance
2. **Canon explanation** — why this is on-style or off-style, grounded in a style constitution
3. **Quality judgment** — approved/rejected with per-dimension scores and cited rules

The output feeds into [`@mcptoolshop/repo-dataset`](https://github.com/mcp-tool-shop-org/repo-dataset) to produce multimodal training data for vision-language model fine-tuning.

## Current lane

**RPG Item Icons** — weapons, potions, scrolls, accessories. 20 subjects x 4 variations = 80 candidates. SDXL Lightning + IconsRedmond LoRA.

## Workflow

```bash
# 1. Generate candidates (requires ComfyUI running)
node scripts/generate.js

# 2. Curate — approve, reject, or mark borderline
node scripts/curate.js iron_sword_base approved "Clean silhouette, correct palette"
node scripts/curate.js iron_sword_icon_mid rejected "3D render look" --failures 3d_render_look
node scripts/curate.js --list  # show uncurated

# 3. Record comparisons
node scripts/compare.js iron_sword_base iron_sword_icon_strong a "Better painterly execution"

# 4. Export training data
npm run export          # paths + image folder
npm run export:embed    # base64-embedded JSONL

# 5. Inspect / validate
npm run inspect
npm run validate
```

## Directory structure

```
workflows/          Versioned ComfyUI workflows
inputs/prompts/     Prompt packs (JSON config for batch generation)
inputs/references/  IP-Adapter reference images
inputs/control/     ControlNet control images
outputs/candidates/ Raw generations (gitignored)
outputs/approved/   Curated good (gitignored)
outputs/rejected/   Curated bad (gitignored)
records/            One JSON per asset (provenance + judgment + canon)
comparisons/        A-vs-B preference judgments
canon/              Style constitution + review rubric
exports/            repo-dataset outputs (gitignored)
scripts/            Automation (generate, curate, compare)
```

## Requirements

- [ComfyUI](https://github.com/comfyanonymous/ComfyUI) running on localhost:8188
- Node.js 20+
- `@mcptoolshop/repo-dataset` for export (optional, via npx)
