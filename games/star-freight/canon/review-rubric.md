# Review Rubric — Gritty Space Uniforms & Costumes

## The Vibe Check (2 seconds)

Does this look like someone who works on a ship for a living? Or does it look like a costume at a convention? If it's the latter, reject.

## Full Review Protocol

1. **Faction ID (instant):** Can you name the faction without reading the filename? If not → likely reject.
2. **Material check:** Does every surface scream the faction's material word? Pressed/Grown/Forged/Woven/Stripped?
3. **Wear check:** Is the aging appropriate? Compact maintained, Veshan battle-scarred, Reach jury-rigged?
4. **Shape language:** Do the silhouette lines match the faction? Compact=rectangles, Veshan=chevrons, Orryn=ellipses?
5. **Color check:** Is the palette muted? Are faction colors present at correct ratios?
6. **Clothing logic:** Do the layers make sense? Does the construction match how that faction builds things?
7. **Generic sci-fi test:** Could this costume appear in ANY sci-fi game/show? If yes, it's not specific enough → reject.
8. **Firefly test:** Would this costume feel at home on Serenity? If it's too clean, too heroic, or too designed → reject.

## Common Failure Modes

| Failure | Rule | How to Spot |
|---------|------|-------------|
| generic_scifi | CLO-003 | Looks like Mass Effect / Star Trek / generic space game |
| too_clean | MAT-002 | Brand new looking, no wear, no history |
| wrong_material | MAT-001 | Veshan wearing woven cloth, Compact wearing forged plate |
| wrong_shapes | SHP-001 | Compact with curves, Veshan with ellipses |
| wrong_palette | COL-001 | Faction colors missing or wrong, bright saturated tones |
| photorealistic | RND-001 | Looks like a photograph, not painterly concept art |
| 3d_render | RND-001 | Too-smooth gradients, shader banding, CG plastic look |
| bad_composition | RND-004 | Not full body, not centered, environmental background |
| costume_not_clothes | CLO-003 | Looks "designed" rather than "owned" — too theatrical |
| faction_unreadable | multiple | Can't tell which faction at a glance |
| anime_style | RND-001 | Big eyes, exaggerated proportions, clean lines |
| hero_pose | RND-004 | Dynamic action pose instead of neutral standing |

## Pairwise Comparison Protocol

When comparing A vs B:
1. Both must be same faction and approximate rank tier
2. Judge which better delivers the gritty, lived-in, faction-specific feel
3. The one that looks more like a real person's work clothes wins over the one that looks more like a costume
4. Cite specific dimensions where one beats the other
